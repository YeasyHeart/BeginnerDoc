# icon批量生成

前期准备

1. 电脑里必须有Photoshop

2. [必须有appIconScript.jsx脚本支持](appIconScript.jsx)

开始动工

1. 打开Photoshop 

2. 把修改好的appIconScript.jsx拖入到Photoshop上

3. 选择图片，选的图片必须是1024x1024尺寸的

4. 选择生成到的目录

PS:修改appIconScript.jsx中的代码，设置你想要的icon名字和大小
